# Extension Icons

This directory should contain the following icon files:
- icon16.png (16x16px)
- icon32.png (32x32px) 
- icon48.png (48x48px)
- icon128.png (128x128px)

You can create simple icons using any image editor or online icon generator.
For now, you can use placeholder icons or create simple ones with a text editor symbol or letter "C" for Cover Letter Generator.
